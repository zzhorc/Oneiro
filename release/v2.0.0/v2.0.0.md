# 🎉 Oneiro v2.0.0

> 品牌重塑：**浮生梦 → Oneiro**（源自希腊语 *ὄνειρος*，意为"梦"）

一个优雅的 Chrome 新标签页扩展 — 在新标签页上展示中国经典诗词，以 Apple 风格展示浏览器书签。

## ✨ 新增功能

- 🔖 **Apple 风格书签栏** — 圆角方形图标展示浏览器书签栏内容
- 📂 **文件夹嵌套展开** — 毛玻璃弹出面板，支持多层递归打开
- 🔄 **实时同步** — 浏览器书签增删改即时反映到新标签页
- ⚙️ **自定义展示行数** — 书签栏 1~4 行可调，设置自动持久化
- 💬 **悬浮提示** — tooltip 显示书签完整名称与网址
- 🏷️ **字体名称提示** — 右下角显示当前使用的字体名称
- 🎨 **全新扩展图标** — 佛系体「梦」字 + 红色圆形背景

## 🔧 优化

- 🔤 文件夹内文字统一使用霞鹜文楷，阅读体验更一致
- 🔠 字体回退链优先使用霞鹜文楷，减少缺字问题
- 🌗 深色 / 浅色主题完整适配

## 📦 安装

| 文件 | 适用浏览器 |
|------|-----------|
| `oneiro-chrome-edge.zip` | Chrome / Edge |
| `oneiro-firefox.zip` | Firefox |

**Chrome / Edge：**
1. 下载并解压 `oneiro-chrome-edge.zip`
2. 打开 `chrome://extensions/` → 开启 **开发者模式**
3. 点击 **加载已解压的扩展程序** → 选择解压后的文件夹

**Firefox：**
1. 下载并解压 `oneiro-firefox.zip`
2. 打开 `about:debugging#/runtime/this-firefox`
3. 点击 **临时载入附加组件** → 选择文件夹中的 `manifest.json`
