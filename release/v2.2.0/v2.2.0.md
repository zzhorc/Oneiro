## 新增特性 (Features)
-  **新增扩展图标popup面板**：方便在任意页面呼出书签和常用网站面板。
-  **多样的书签视图矩阵**：
    -   **新增「杂志视图」**：单行两列展示。
    -   **新增「列表视图」**：单行排满，信息密度高。
    -   *注：你可以在 Popup 面板右上角随时无缝切换这三个视图，且嵌套的文件夹将继承你选择的排版。*
-  **独立的 Popup 展现逻辑**：点击扩展触发的 Popup 弹窗现在永远会展现你的所有数据，不受主页「设置」中隐藏书签或常用网站的干扰。

## 修复 (Bugfixes)
-  ** Chrome Storage 原生数据迁移**：避免当用户手动覆盖更新插件时「书签与常用网站丢失」，现在可以平滑过渡，所有历史网址都会转移至永久安全的 Chrome 独立物理存储沙盒中。
